# Datos del Proyecto

## Dataset Requerido

Coloca el archivo `hospital_readmissions.csv` en esta carpeta.

### Cómo obtenerlo

**Opción A — UCI ML Repository:**
1. Visita: https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008
2. Descarga el archivo ZIP y extrae `hospital_readmissions.csv`.

**Opción B — Kaggle:**
https://www.kaggle.com/datasets/dubradave/hospital-readmissions

### Descripción del Dataset

- **Registros:** 25 000 pacientes hospitalizados
- **Variables:** 17 (16 predictoras + 1 objetivo)
- **Variable objetivo:** `readmitted` (yes / no)
- **Distribución de clases:** 53 % no readmitidos / 47 % readmitidos
- **Fuente original:** Strack et al. (2014). UCI ML Repository.
